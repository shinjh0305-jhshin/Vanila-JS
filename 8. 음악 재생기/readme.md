음악 재생기
==================
재생, 일시정지, 되감기, 빨리감기 기능이 있는 음악 재생기를 구현합니다.
* AUDIO API를 이용해서 구현합니다.
## 추후 개선 사항
    1. 오디오 플레이리스트 구현해서 현재 재생중인 음악 표시.
    2. 현재 시각, 총 시각 표시.(원본 코드 참조)
