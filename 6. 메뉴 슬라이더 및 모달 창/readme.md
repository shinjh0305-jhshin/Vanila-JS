메뉴 슬라이더 및 모달 창
==================
버튼을 클릭 해 메뉴 슬라이더 및 모달 창을 표시하며, 임의의 공간을 클릭해서 열었던 창을 닫습니다.
* 외부 라이브러리 및 문서를 이용할 때, 코드의 변조를 검사하는 SubResource Integrity(SRI)를 적용합니다.
* 모달창과 팝업 창의 구현 상 차이를 이해합니다.
